Ejemplos de Angular con Bootstrap

Interpolación

El componente app-interpolacion muestra un mensaje interpolado en el HTML. El mensaje se obtiene de un atributo de entrada llamado mensaje y se muestra en el template.

Property Binding

El componente app-property-binding muestra una imagen en el HTML. La URL de la imagen se obtiene de un atributo de entrada llamado imagenUrl y se muestra en el template.

ngStyle

El componente app-ngstyle muestra un mensaje en el HTML. El mensaje se obtiene de un atributo de entrada llamado mensaje y se muestra en el template. Además, el mensaje se muestra con un estilo personalizado.

ngClass

El componente app-ngclass muestra un mensaje en el HTML. El mensaje se obtiene de un atributo de entrada llamado mensaje y se muestra en el template. Además, el mensaje se muestra con una clase personalizada.

El estilo de la clase se define en el archivo styles.css en la carpeta src/styles.

El estilo se aplica al mensaje utilizando la directiva ngClass.
